# Qual a Boa?
Plataforma web para descobrir locais de lazer em Brasília.

## 🚀 Tecnologias
- PHP
- MySQL
- HTML, CSS, JavaScript

## 🛠 Como rodar
1. Clone o repositório
2. Importe `db/script.sql` no MySQL
3. Configure o `config.php` com as credenciais do banco
4. Inicie o servidor local (XAMPP, WAMP, etc.)
